# Echo 项目规则

## 修改 Echo.html 前的备份规则

每次修改 `e:\项目\Echo\Echo.html` 之前,必须先将当前版本复制一份到 `e:\项目\Echo\副本\` 目录中。

**命名格式**:`版本号_修改内容.html`(例:`v1.0_修复首页轮播图.html`、`v1.1_新增暗黑模式.html`)

**版本号规则**:
- 首次备份为 `v1.0`
- 同一类小修改递增小数位:`v1.1`、`v1.2`
- 大改或重构时主版本号递增:`v2.0`、`v3.0`

**操作步骤**:
1. 查看 `e:\项目\Echo\副本\` 目录中已有的备份,确定下一个版本号
2. 使用 `copy` 命令将 `Echo.html` 复制为 `e:\项目\Echo\副本\版本号_修改内容.html`
3. 确认备份成功后再开始修改原文件

## 界面设计规则

- 不要使用蓝紫色调
- 不要使用 emoji 表情
